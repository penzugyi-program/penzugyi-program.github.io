2020. május. 27. 8:50 HVG

A magyar állampolgároknak készült pénzügyi program az enyhített korlátozások idején is tovább működik. Kérjük, olvassa el a követelményeket.

Az enyhített karantén korlátozások idején a speciális pénzügyi program tovább fog működni. A magyar állampolgárok akár 1,000 eurót kaphatnak hetente fejenként.

05.15. és 06.15. között a magyar állampolgárokat arra kérik, a program hivatalos weboldalán kérdéseket válaszoljanak meg.

Tovább a weboldalra

Ha a program követelményeinek megfelel, megkapja a lehetőséget, hogy meghallgasson egy 30 perces hangüzenetet, olyan utasításokkal, amik segítenek akár heti 1,000 euró extra jövedelemhez jutni. A program magyar szervezői és szponzorai állítása szerint a hívás fizetős, mert így korlátozzák koron aluli személyek és informatikusok hozzáférését, akik kiskapukat keresnek a program algoritmusában, hogy több személynek is kiutalhassák a pénzt.

A pénzügyi program első napján sok negatív hozzászólás érkezett olyan emberektől, akiknek pénzügyi támogatásra volt szükségük, de a programra nem nyertek felvételt. A program szervezői azt nyilatkozták, a programba naponta csak korlátozott számban csatlakozhatnak. Ha reménytelenül szüksége van pénzre, próbálja újra másnap, mert akkor valószínűleg felvételt nyer a programra.
